# Premium Dashboard Upgrade

Implemented a matured premium user dashboard for Hey Larmah Enterprise Limited.

## Included
- Premium overview with four pillar counters.
- Structured Request Centre for Real Estate, Fintech, Logistics, Shipping and Premium support.
- Pillar workspaces with desk-specific expectations and quick request buttons.
- Request tracking cards with statuses and next actions.
- Document Vault with file upload support through Supabase Storage.
- Premium catalogue preview loaded from Supabase.
- Support desk, account security, reset, reauthentication and email change flows retained.
- Admin Client Request Desk for updating status, next action and admin notes.
- Visibility CSS blended into the live stylesheet while preserving Hey Larmah brand colors.

## Before going live
Run `schema.sql` in Supabase SQL Editor so the `request_documents` table, dashboard request statuses and storage policies are available.
